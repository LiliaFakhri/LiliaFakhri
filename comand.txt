1. Склонировать репозиторий на свой компьютер в отдельную папку
git clone SSH ссылка на GitHub репозиторий
2.Проверить статус изменений
Get status
3. Добавить изменения
Git add -a
4.Добавить комит 
git commit -m "message"
5.Отправить изменения в удаленный репозиторий
git push origin main
6.Подтянуть изменения из удаленного в локальный репозиторий
git pull
